1️⃣  INSTALL DEPENDENCIES
   ─────────────────────────────────────────────────────────────────

2️⃣  TEST NEW FEATURES (Optional but Recommended)
   ─────────────────────────────────────────────────────────────────

   python test_new_features.py
   
   You should see:
   ✅ Risk Calculator Tests Passed!
   ✅ Hospital Finder Tests Passed!
   ✅ Integration Tests Passed!
   ✅ ALL TESTS PASSED SUCCESSFULLY!


3️⃣  START THE APPLICATION
   ─────────────────────────────────────────────────────────────────
   Run the main application:
   
   python app.py
   
   You should see:

4️⃣  OPEN IN BROWSER
   ─────────────────────────────────────────────────────────────────

   http://localhost:5002
   
   Or click the link in the terminal

   ─────────────────────────────────────────────────────────────────

      • Download PDF report
   
   D. CREATE DOCTOR ACCOUNT:
      • Sign up with Role: Doctor
      • Login to see Analytics Dashboard
      • View all charts and statistics
      • Check high-risk patients table
      • Review age-wise disease distribution

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

PROBLEM: Dependencies not installing
SOLUTION: Make sure you have Python 3.8+ installed
          Try: pip install --upgrade pip
          Then: pip install -r requirements.txt

PROBLEM: Port 5002 already in use
SOLUTION: Change port in app.py (last line):
          app.run(debug=True, port=5003)

PROBLEM: Database errors
SOLUTION: Delete healthcare_system.db file
          Restart the application (it will recreate)

PROBLEM: Charts not showing
SOLUTION: Check internet connection (Chart.js loads from CDN)
          Clear browser cache
          Try different browser

PROBLEM: Google Maps not opening
SOLUTION: Check if popup blocker is enabled
          Allow popups for localhost
          Try right-click → Open in new tab
