📞 SUPPORT & TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Common Issues:

Issue: Dependencies not installing
Fix: pip install --upgrade pip
     pip install -r requirements.txt

Issue: Port already in use
Fix: Change port in app.py (last line)
     app.run(debug=True, port=5003)

Issue: Charts not showing
Fix: Check internet connection (Chart.js CDN)
     Clear browser cache

Issue: Google Maps not opening
Fix: Allow popups for localhost
     Check popup blocker settings

For More Help:
• Check RUN_INSTRUCTIONS.txt
• Run test_new_features.py
• Review error messages in terminal
• Check browser console
