❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Q: App won't start?
A: Make sure you're in the right folder:
   cd Smart_Healthcare_System/Enhanced_Healthcare_System
   python app.py

Q: Import errors?
A: Check that these files exist:
   - risk_calculator.py
   - hospital_finder.py
   - medicine_db.py
   - model.py

Q: Database errors?
A: The old database was deleted. It will be recreated automatically
   when you run the app for the first time.

Q: No hospitals showing?
A: Make sure you:
   1. Selected a city during signup (try "New York")
   2. Didn't set distance filter too low
   3. Try removing filters (leave blank)

Q: Charts not loading?
A: Check internet connection (Chart.js loads from CDN)
   Clear browser cache and refresh

Q: Risk score seems wrong?
A: Make sure you:
   1. Filled in lifestyle info during signup
   2. Selected all your symptoms
   3. Have some past history (affects score)
