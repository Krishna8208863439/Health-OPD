THE ROOT CAUSE DISCOVERED:
═══════════════════════════════════════════════════════════════════════════════

The training data uses an INVERTED logic:
• 1 = HAS the symptom ✓
• 2 = DOESN'T have the symptom ✗

This is OPPOSITE of what we expected!

I was converting:
  Checked (1) → 2 (thinking 2 = has symptom) ❌ WRONG!
  Unchecked (0) → 1 (thinking 1 = no symptom) ❌ WRONG!

Correct conversion:
  Checked (1) → 1 (1 = HAS symptom) ✅ CORRECT!
  Unchecked (0) → 2 (2 = NO symptom) ✅ CORRECT!

  Swollen feet average: 1.00 ← Always 1 = they HAVE swollen feet!


THE FIX:
═══════════════════════════════════════════════════════════════════════════════

Changed the conversion function:

BEFORE (Wrong):
    def convert_symptom(value):
        return 2 if int(value) == 1 else 1  ❌

AFTER (Correct):
    def convert_symptom(value):
        return 1 if int(value) == 1 else 2  ✅
