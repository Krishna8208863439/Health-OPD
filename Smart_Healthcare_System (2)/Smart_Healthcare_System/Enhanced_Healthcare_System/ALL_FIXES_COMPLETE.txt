FIX #2: Symptom Value Conversion
────────────────────────────────────────────────────────────────────────────
The ML model expects:
• Symptoms: 1 (no symptom) or 2 (has symptom)
• Fever: 97-105 (temperature values)

But checkboxes send:
• 0 (unchecked) or 1 (checked)

Solution: Convert checkbox values to model format
    Unchecked (0) → 1 (no symptom)
    Checked (1) → 2 (has symptom)
    Fever checked → 102 (fever temperature)
    Fever unchecked → 98 (normal temperature)

Result: Model now predicts different diseases correctly
