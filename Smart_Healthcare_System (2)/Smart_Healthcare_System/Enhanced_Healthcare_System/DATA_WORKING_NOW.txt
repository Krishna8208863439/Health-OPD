❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Q: App won't start?
A: Make sure you're in the right folder:
   cd Smart_Healthcare_System/Enhanced_Healthcare_System
   python app.py

Q: Templates not showing new features?
A: Make sure all template files were updated:
   - signup.html (lifestyle fields)
   - symptom_entry.html (hospital filters)
   - result_enhanced.html (risk score)
   - dashboard_enhanced.html (analytics)

Q: Risk score not showing?
A: Make sure you:
   1. Filled in lifestyle info during signup
   2. Have risk_calculator.py in the folder
   3. App is using result_enhanced.html template

Q: No hospitals showing?
A: Make sure you:
   1. Selected a city during signup (try "New York")
   2. Have hospital_finder.py in the folder
   3. Try removing hospital filters

Q: Charts not loading in dashboard?
A: 1. Check internet connection (Chart.js CDN)
   2. Clear browser cache
   3. Check browser console for errors

Q: PDF not downloading?
A: 1. Check that static/reports directory exists
   2. Check console for PDF generation messages
   3. Run test_pdf_generation.py
